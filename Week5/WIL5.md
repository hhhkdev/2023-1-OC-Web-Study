# **CSS**
Cascading Style Sheet

## **1. 상속**
부모태그와 자식태그 사이의 스타일 상속 관계

## **2. 우선순위**

### **스타일 시트**

1. 사용자 스타일(컴퓨터 사용자)
2. 제작자 스타일(개발자)
3. 브라우저 기본 스타일

```
1. !important
2. 인라인 스타일
3. id 스타일
4. 클래스 스타일
5. 타입 스타일
```
-> Selector를 통해 지정 가능

# Selector
`#id {}` id 선택자

`.class {}` class 선택자

`<tag_name> {}` 타입 선택자

`* {}` 전체 선택자
